##     Project Walkthrough

### Phase 1: Data Understanding
- **5,000 customers** with 22 features
- **34.6% historical churn rate**
- Features include: demographics, contract details, services, usage, satisfaction

### Phase 2: Exploratory Analysis
**Key Findings:**
- New customers (0-6 months) churn at 59.2%
- Month-to-month contracts: 43.7% churn
- Low satisfaction (<3): 38-58% churn

### Phase 3: Model Development
**Trained 3 models:**
- Logistic Regression     (Best: 77.8% ROC-AUC)
- Random Forest (75.1% ROC-AUC)
- Gradient Boosting (76.8% ROC-AUC)
